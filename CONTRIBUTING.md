# 贡献指南

谢谢你愿意补充长寿经济产业图谱。这个项目欢迎企业、园区、研究者、开发者和关心健康寿命的人一起维护，但所有新增内容都应尽量可追溯、可验证、可继续追问。

## 基本原则

- 优先使用官方介绍、政府文件、企业公告、注册上市信息、临床试验登记、年报、融资公告和权威媒体报道。
- 区分事实、判断和待验证问题。没有来源的内容请写成“待验证”，不要写成事实。
- 医疗、金融和政策相关内容只作产业研究与资源导航，不构成医疗、投资或法律建议。
- 每条记录尽量保留来源链接、更新时间和贡献者。

## 审阅清单

提交前请确认：

- 是否能看出这条内容属于长寿经济的哪个产业节点？
- 是否解释了它和健康寿命延长的关系？
- 是否提供了至少一个公开来源？
- 是否把不确定信息写成待验证问题？
- 是否避免了医疗、投资或法律建议式表达？

## 推荐贡献路径

小补充可以直接提交 pull request；较大的城市、园区或专题案例，建议先使用 GitHub Issue 模板说明资料范围、来源和计划补充的字段。

- 产品案例：https://github.com/proteinb/longevity-economy-atlas/issues/new?template=product-case.md
